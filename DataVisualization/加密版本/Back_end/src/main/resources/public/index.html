宝贝回家 404 页面介绍

宝贝回家 404页面是由 uiuing 发起的互联网公益活动。

网站开发者只需要在自己的404页面中嵌入一段简单的代码，就能通过互联网来迅速传播失踪儿童信息，从而提高找回失踪儿童的概率。失踪儿童信息采集自宝贝回家寻子网。

宝贝回家 404页面的特别之处

在访问者看到失踪儿童的信息之前，无论是这些信息的收集还是筛选，我们都进行了大量的工作，除此之外，我们还进行了十分深入的数据挖掘与分析，在我们后台的的预测模型计算之下，宝贝回家 404页面将尽可能推算出地访问者身边的失踪儿童。

尽管这点工作微不足道，但也希望能在公益行动之中，我们在尽可能的争取更好。
